** **Yoyo fruity yogurt****
A yogurt website created for a school business presentation 

This project is built primarily using HTML, CSS, and vanilla JavaScript. Its main purpose is to demonstrate the beauty of web design 

**FEATURES**
* Feature 1 : Displays a dynamic product menu sourced from a JavaScript array
* Feature 2  :Allows users to add and remove items from a shopping cart
* Feature 3 : Calculates the subtotal and applies a delivery fee
* Feature 4 :Uses modal windows for confirmation messages

 ****Technologies Used**
* *HTML5:* For the structural foundation.
* *CSS3:* For styling and responsiveness.
* *JavaScript (ES6+):* For all interactive logic, product data management, and DOM manipulation.


> This project was created as part of a school's entrepreneurship presentation
